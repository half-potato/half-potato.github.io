EIGEN_STRONG_INLINE Scalar* Sophus::SE2Group< _Scalar, _Options >::data ( )
inline
Returns
pointer to internal data

This provides unsafe read/write access to internal data. SE2 is represented by a pair of an SO2 element (two parameters) and a translation vector (two parameters). The user needs to take care of that the complex stays normalized.

/see normalize()
